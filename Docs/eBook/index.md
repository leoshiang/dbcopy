**作者備註**

本書以 DbCopy 系統的實際原始碼為基礎，從設計動機到實作細節完整呈現。
所有程式碼範例均取自真實的 production codebase，而非簡化的教學版本。

C# 程式碼採用全中文命名，資料庫維持英文命名——這是本專案的設計準則。

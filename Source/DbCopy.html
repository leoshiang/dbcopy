核心功能

一站式資料庫複製解決方案

Web UI 操作介面

內建 Razor Pages 網頁介面，透過瀏覽器即可完成資料庫比對與複製，操作直覺簡單。

REST API

完整的 RESTful API 端點，可整合至 CI/CD 流程或自動化腳本，實現程式化操作。

結構比對

自動比對來源與目標資料庫的結構差異，識別缺少的資料表、檢視、預存程序等物件。

四階段複製

結構建立、資料搬移、索引建立、外鍵約束，分階段執行確保資料完整性與正確性。

單一執行檔

發布為 Self-contained 單一檔案，內建壓縮，無需安裝 .NET Runtime 即可執行。

跨平台支援

原生支援 Windows、Linux (x64/ARM64)、macOS (Intel/Apple Silicon)，一次編譯到處執行。

相依性解析

自動分析物件之間的相依關係，決定正確的建立順序，避免建立失敗。

完整日誌

採用 Serilog 結構化日誌，記錄每一條 SQL 語句，方便除錯與稽核。

四階段複製流程

確保結構與資料的完整遷移

1
結構建立

建立資料表結構、欄位定義、主鍵與約束條件。

2
資料搬移

使用 Bulk Copy 高效搬移資料，支援批次處理。

3
索引建立

建立次要索引，提升查詢效能。

4
外鍵約束

加入外鍵約束，確保參考完整性。

REST API

完整的 RESTful 端點，支援程式化操作

端點 方法 說明
/api/db/test POST 測試資料庫連線是否正常
/api/db/objects POST 取得資料庫物件清單（資料表、檢視、預存程序等）
/api/db/compare POST 比對來源與目標資料庫結構差異
/api/db/copy POST 執行物件與資料複製

詳細規格

使用教學與技術規格

執行 DbCopy 後，開啟瀏覽器即可使用 Web UI。

$ ./DbCopy
// 瀏覽器開啟 http://localhost:5281

透過 API 測試資料庫連線。

$ curl -X POST http://localhost:5281/api/db/test \
  -H "Content-Type: application/json" \
  -d '{
    "type": 0,
    "connectionString": "Server=localhost;Database=MyDb;Trusted_Connection=true;"
  }'

列出資料庫中的所有物件。

$ curl -X POST http://localhost:5281/api/db/objects \
  -H "Content-Type: application/json" \
  -d '{
    "type": 0,
    "connectionString": "Server=localhost;Database=MyDb;Trusted_Connection=true;"
  }'

比對來源與目標資料庫，找出差異。

$ curl -X POST http://localhost:5281/api/db/compare \
  -H "Content-Type: application/json" \
  -d '{
    "source": { "type": 0, "connectionString": "..." },
    "target": { "type": 0, "connectionString": "..." }
  }'

將指定物件從來源複製到目標資料庫。Phase 0 表示執行全部階段。

$ curl -X POST http://localhost:5281/api/db/copy \
  -H "Content-Type: application/json" \
  -d '{
    "source": { "type": 0, "connectionString": "..." },
    "target": { "type": 0, "connectionString": "..." },
    "object": { "schema": "dbo", "name": "Users", "type": "Table" },
    "phase": 0,
    "batchSize": 1000
  }'

執行 publish.sh 可輸出 Windows/macOS/Linux 到 publish/ 目錄。

$ ./publish.sh
